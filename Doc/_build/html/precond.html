precond module

precond.Px(Ix2, Iy2, lmbda, lmbda2, N, M, x)

This function is used to compute the product of P^-1*x without storing the matrix P^-1. Where P is the given preconditioner and x is a given vector. This function will be used to construct the preconditioned version of the Minres solver.

Parameters:

Ix2array

The elemnt-wise square of the image derivative with respect to x

Iy2ndarray

The elemnt-wise square of the image derivative with respect to y

lmbdafloat

The parameter of Tikhonov

lmbda2float

A weight used to take into account the non local term

Nint

The number of the image rows

Mint

number of the image cols

xndarray

A 2*M*N given vector

Returns:

res : ndarray A 2*M*N vector res=P^-1*x
